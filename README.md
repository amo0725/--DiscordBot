# ChiBot
"吉器人" 是 Discord 的機器人 !  

# 普通指令

查詢帳戶資訊

```
!餘額
```

存款機

```
!存 金額
```

提款機

```
!提 金額
```

轉帳機

```
!轉 @誰 金額
```

簽到領獎 ( 一天只能一次，每次 800 元 )

```
!簽到
```

上街乞討 ( 每次獲取 1 - 100 元 不等)

```
!乞討
```

賭硬幣

```
!丟 正or反 金額
```

賭骰子

```
!骰 點數 金額
```

拉霸機

```
!拉 金額
```

挖礦 ( 50  /次 )

```
!挖 次數
```

查詢背包物品

```
!背包
```

賣出背包物品

```
!賣 物品名稱 數量
```

查詢所有指令

```
!指令
```


# 音樂指令

撥放

```
$播 網址or歌曲名
```

跳到下一首

```
$跳
```

停止撥放

```
$停
```

顯示撥放列表

```
$列
```

跳到某一首

```
$跳到 num
```
